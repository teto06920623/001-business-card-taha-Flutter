## ✨ Overview

This project is a simple yet polished digital business card application that showcases personal identity and contact details in a modern mobile UI.

It demonstrates:

- A centered profile layout
- Circular avatar presentation
- Name and professional title styling
- Contact information cards
- Clean Material Design implementation
- Responsive UI across platforms

## 🏁 Getting Started

### Prerequisites

Make sure you have the following installed:

- Flutter SDK
- Dart SDK
- Android Studio / VS Code
- An emulator, simulator, or physical device

### Installation

```bash
git clone https://github.com/teto06920623/001-business-card-taha-Flutter.git
cd 001-business-card-taha-Flutter
flutter pub get
```

### Run the App

```bash
flutter run
```

---

## 🧪 Build for Release

### Android

```bash
flutter build apk --release
```

### iOS

```bash
flutter build ios --release
```

### Web

```bash
flutter build web
```

## 📌 Notes

- Ensure your asset paths are correctly declared in `pubspec.yaml`
- If you update the profile image, verify the new asset is included in the project
- Use `flutter clean` if you encounter build issues after changes
