# 🆔 Personal Business Card App

[![Flutter](https://img.shields.io/badge/Flutter-02569B?logo=flutter&logoColor=white)](https://flutter.dev/)
[![Dart](https://img.shields.io/badge/Dart-0175C2?logo=dart&logoColor=white)](https://dart.dev/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/Platform-Android%20%7C%20iOS%20%7C%20Web-blue)](https://flutter.dev/multi-platform)

A clean and professional **digital business card app** built with **Flutter**.  
This project demonstrates a simple yet elegant UI using Flutter’s layout widgets, custom styling, and card-based contact presentation.

---

## ✨ Overview

This app presents a personal identity card featuring:

- A profile image in a circular avatar
- Name and professional title
- Contact details in polished card layouts
- A minimalist, responsive, and centered design

---

## 🚀 Features

| Feature | Description |
|--------|-------------|
| 👤 Profile Section | Displays a circular profile image using `CircleAvatar` |
| 🎨 Custom Styling | Stylish typography and color palette for a modern look |
| 📞 Contact Cards | Clean phone and email display using `Card` and `ListTile` |
| 📱 Responsive UI | Built to look good across different screen sizes |
| 🧱 Flutter Widgets | Uses core widgets like `SafeArea`, `Column`, `Card`, and `ListTile` |

## 📦 Getting Started

### Prerequisites
Make sure you have the following installed:

- Flutter SDK
- Dart SDK
- Android Studio / VS Code
- An emulator or physical device

### Installation

```bash
# Clone the repository
git clone https://github.com/teto06920623/001-business-card-taha-Flutter.git

# Navigate to the project directory
cd 001-business-card-taha-Flutter

# Get dependencies
flutter pub get

# Run the app
flutter run
```
